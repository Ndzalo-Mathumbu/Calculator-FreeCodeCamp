# JavaScript Calculator

A responsive calculator application built with **React** and **Sass** as part of the **FreeCodeCamp Frontend Libraries Certification**.

The calculator performs basic arithmetic operations, supports decimal numbers, handles chained calculations, and follows the project requirements defined by FreeCodeCamp.

## Features

* Addition, subtraction, multiplication, and division
* Decimal number support
* Clear all functionality
* Backspace/delete functionality
* Responsive mobile-first design
* Real-time display updates
* Operator chaining support
* Error handling for invalid input
* Built with React state management

## Technologies Used

* React
* JavaScript (ES6+)
* Sass (SCSS)
* HTML5
* CSS3
